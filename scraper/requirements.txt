# Runtime dependencies for the tr-scraper sidecar.
#
# pytr itself is installed separately and pinned exact in the Dockerfile with
# --no-deps, so the dep set below is fully curated. The list is pytr 0.4.9's own
# dependency set, plus the web server, plus cloakbrowser + playwright: TR now
# gates the login behind AWS WAF Bot Control, so we mint a valid `aws-waf-token`
# with a real stealth Chromium (app/waf.py) and hand it to pytr. The real
# playwright (a cloakbrowser dependency) also satisfies pytr's module-level
# `playwright` import — the old vendored stub is gone.
#
# websockets is pinned >=16 deliberately: squid answers the proxy CONNECT with
# HTTP/1.0, and websockets <16 rejects HTTP/1.0 proxy replies, which would break
# the wss://api.traderepublic.com path.

cloakbrowser
playwright

babel
certifi
coloredlogs
cryptography
curl-cffi
packaging
pathvalidate
pygments
requests-futures
shtab
websockets>=16,<17

fastapi>=0.115
